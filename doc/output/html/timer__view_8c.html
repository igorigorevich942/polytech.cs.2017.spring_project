Создание отображения таймеров.

Аргументы
[in]t_model- адрес модели таймеров.
[in]width- ширина области рисования.
[in]height- высота области рисования.
[in]shift_x- смещение позиций рисования таймеров по оси OX.
Возвращает
Адрес, созданного отображения таймеров.
NULL, если во время создания отображения таймеров произошла ошибка.
